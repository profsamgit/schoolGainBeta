# **App Name**: SchoolGain Hub

## Core Features:

- User Authentication & Profiles: Secure login for students and educators with Firebase Authentication. Display user profiles, including 'Agente da Sustentabilidade' roles for students, and track basic stats like total points.
- Manual Waste Registration: Allow students to manually record their waste disposals, selecting the waste type (plastic, paper, metal, organic). This simulates initial Lixeira Inteligente integration, providing immediate point feedback.
- Gamification Progress Tracking: Display user's accumulated points, current achievement level (e.g., Bronze, Silver), and a simplified leaderboard for individual students to foster healthy competition.
- Educational Content Viewer: Browse pre-defined environmental educational content modules including text, images, and embedded videos to align with BNCC guidelines.
- AI-Powered Educational Quizzes: Generate dynamic quizzes based on selected environmental topics. This tool will test students' knowledge and award points for correct answers, making learning interactive and engaging.
- Rewards Catalogue & Redemption: Display a catalogue of available rewards (e.g., school supplies, special privileges). Students can view and redeem rewards using their accumulated points.
- School Administrator Dashboard: A simplified dashboard for school administrators to view aggregate data, such as total waste collected by type and overall student engagement statistics (e.g., top contributors).

## Style Guidelines:

- Color Palette: A light, clean aesthetic reflecting sustainability and growth, designed for readability and engagement.
- Primary Color: A lively, refreshing green (#22C322) to symbolize nature, growth, and environmental responsibility, making it suitable for headings and interactive elements.
- Background Color: A very light, almost white, subtle green (#EFF7EF) to maintain a clean interface and allow the primary color to stand out.
- Accent Color: A vibrant cyan-green (#4DCC8C) for highlighting key information, call-to-action buttons, and progress indicators, creating good contrast and visual interest.
- Headline and Body Font: 'Poppins' (sans-serif) for its geometric, modern, and engaging appearance, appealing to children and adolescents and suitable for a gamified experience.
- Icon Style: Utilize playful and clear icons and badges for gamification elements (e.g., achievements, waste categories), with a clean and flat design to complement the overall aesthetic.
- Responsive Design: An adaptive layout based on a card-based structure, ensuring a consistent and user-friendly experience across web and mobile devices. Emphasize clear hierarchy and visual separation.
- Gamification Animations: Incorporate subtle, delightful animations for point accumulation, level-ups, and successful actions to enhance the sense of achievement and user engagement.
